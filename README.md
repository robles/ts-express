# ts-express
Express api using typescript compiled by babel
- hot reload using nodemon
- Jest for testing
- Typescript compiler for type checking

## Project setup
```
npm install
```

### Starts and hot reloads for development
```
npm start
```

### Builds for production
```
npm run build
```

### Run unit tests
```
npm test
```

### Run linter
```
npm run lint
```
